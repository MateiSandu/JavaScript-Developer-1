Variables

The name of a variable is called an identifier, and it must conform to certain naming rules. A variable can be declared using the var, let, or const keyword. Global variables exist outside any function while local variables are defined inside a particular function.

Examples:

var globalVar = 10; // Global variable
let localVar = 'Local'; // Local variable
const PI = 3.14; // Constant variable

Data Types

JavaScript supports eight data types for storing data:

  • Boolean: Represents a logical value: true or false.
  • Number: Represents numeric values, both integers and floating-point numbers.
  • String: Represents a sequence of characters.
  • BigInt: Represents integers with arbitrary precision.
  • Symbol: Represents unique and immutable values.
  • Object: Represents collections of data or complex entities.
  • null: Represents the intentional absence of any value.
  • undefined: Represents a variable that has not been assigned a value.

Examples:

let isTrue = true;
let numberValue = 42;
let stringValue = 'Hello, World!';
let bigIntValue = 9007199254740991n;
let symbolValue = Symbol('foo');
let objectValue = { key: 'value' };
let nullValue = null;
let undefinedValue;

Wrapper Objects

Primitive data types like strings, booleans, and numbers have corresponding wrapper objects that allow them to be treated as objects:

let str = 'Hello';
let strObject = new String('Hello');
console.log(typeof str); // string
console.log(typeof strObject); // object

Typecasting

Typecasting allows you to convert a value from one data type to another:

let numStr = '42';
let num = Number(numStr); // Explicit conversion to number
let boolValue = Boolean(''); // Implicit conversion to boolean (false)
let strValue = String(123); // Implicit conversion to string
